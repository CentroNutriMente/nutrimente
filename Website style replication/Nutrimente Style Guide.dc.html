Design Style Guide

A calm, botanical visual language for the psychology & nutrition studio dashboard — watercolour washes, hand-drawn foliage, an elegant serif paired with a soft humanist sans. Everything below is ready to hand to Claude Code.

01 — Colour

Palette

Warm ivory ground, muted sage as the action colour, soft lavender as accent, gentle blush for warmth. Keep saturation low — nothing shouts.

02 — Texture

Watercolour washes

Soft, blurred radial blooms behind the header and in empty corners. Layer two hues, keep them low-opacity, and always blur. Built purely in CSS — no image assets.

04 — Illustration

Botanical motifs

Recreated as clean, single-stroke SVG line-art using currentColor so each motif recolours to sage, lavender or blush in context. Use them small and sparing — corners, beside greetings, behind empty states.
